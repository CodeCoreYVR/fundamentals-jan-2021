Font-Size

Absolute Units

1 px is 1/96th of an inch, not the pixel of the screen of the viewing device

Half a centimetre font-size. Don't have spaces in-between the values and the CSS unit like this: "0.5 cm", rather this: "0.5cm"

Relative Units

em is relative to the font-size of the parent element if we're using it for font properties. If we're using em for anything that's not font related, then its relative to this element's font-size. This element's font-size is 0.5em x 40px = 20px The border (non-font property) is: 0.25em x (0.5em x 40px) 0.25em x 20px 5px

If the root element's (html element) font-size 16px, then 1.5rem x 16px = 24px

Width

200px wide width only applies to the content, not the paddings or the margins
Percentage units are relative to the parent element's value of the same property 50% of 400px = 200px
vw is relative to the viewport's width, range is from 0 - 100 1 unit of vw is 1% of the viewport's width

Height

20px high
Percentage units are relative to the parent element's value of the same property 50% of 100px = 50px
vh is relative to the viewport's height, range is from 0 - 100 1 unit of vh is 1% of the viewport's height
